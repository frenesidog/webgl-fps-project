# WebGL FPS Game

## Overview
This project is a first-person shooter (FPS) game developed using WebGL. The game leverages the capabilities of WebGL to render 3D graphics directly in the browser without the need for additional plugins.

## Features
- Stunning 3D graphics powered by WebGL
- Realistic physics and collision detection
- Multiple levels with increasing difficulty
- Sound effects and background music
- Player customization options
- High-performance rendering for a smooth gaming experience

## Controls
- **W**: Move Forward
- **S**: Move Backward
- **A**: Move Left
- **D**: Move Right
- **Space**: Jump
- **Mouse Movement**: Look Around
- **Left Click**: Shoot

## How to Run the Project
1. **Clone the repository**:
   ```bash
   git clone https://github.com/frenesidog/webgl-fps-project.git
   cd webgl-fps-project
   ```

2. **Open the index.html file in your browser**:
   Simply double-click on the `index.html` file or use a local server to host the files.

3. **Enjoy the game!**
